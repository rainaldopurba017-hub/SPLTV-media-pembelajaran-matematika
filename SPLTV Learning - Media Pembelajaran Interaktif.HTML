Pendahuluan Konsep SPLTV

Sistem Persamaan Linear Tiga Variabel adalah sistem yang terdiri dari tiga persamaan linear dengan tiga variabel yang tidak diketahui

Variabel Pertama

Bisa mewakili jumlah barang, jarak, harga, dll

Soal Cerita:

Sebuah toko kue membuat 3 jenis kue: A, B, dan C. Untuk membuat kue-kue tersebut diperlukan:

‚Ä¢ Hari pertama: 2 kue A + 3 kue B + 1 kue C = 23 kg bahan
‚Ä¢ Hari kedua: 1 kue A + 2 kue B + 3 kue C = 22 kg bahan
‚Ä¢ Hari ketiga: 3 kue A + 1 kue B + 2 kue C = 24 kg bahan

Berapa kg bahan yang diperlukan untuk membuat masing-masing jenis kue?

Penyelesaian:

Langkah 1: Pahami Masalah

x = bahan untuk kue A (kg)
y = bahan untuk kue B (kg)
z = bahan untuk kue C (kg)

Langkah 2: Model Matematika
2x + 3y + z = 23
x + 2y + 3z = 22
3x + y + 2z = 24
Penyelesaian Lengkap:
Langkah 3: Eliminasi Variabel

Eliminasi x dari persamaan (1) dan (2):
2x + 3y + z = 23 ... (1)
x + 2y + 3z = 22 ... (2)

Kalikan persamaan (2) dengan 2:
2x + 4y + 6z = 44 ... (2')

Kurangkan (1) dari (2'):
(2x + 4y + 6z) - (2x + 3y + z) = 44 - 23
y + 5z = 21 ... (4)
Eliminasi x dari persamaan (2) dan (3):
x + 2y + 3z = 22 ... (2)
3x + y + 2z = 24 ... (3)

Kalikan persamaan (2) dengan 3:
3x + 6y + 9z = 66 ... (2'')

Kurangkan (3) dari (2''):
(3x + 6y + 9z) - (3x + y + 2z) = 66 - 24
5y + 7z = 42 ... (5)
Selesaikan sistem 2 variabel:
y + 5z = 21 ... (4)
5y + 7z = 42 ... (5)

Dari persamaan (4): y = 21 - 5z
Substitusi ke persamaan (5):
5(21 - 5z) + 7z = 42
105 - 25z + 7z = 42
105 - 18z = 42
-18z = -63
z = 3.5
Cari nilai y dan x:
z = 3.5
y = 21 - 5(3.5) = 21 - 17.5 = 3.5

Substitusi y dan z ke persamaan (2):
x + 2(3.5) + 3(3.5) = 22
x + 7 + 10.5 = 22
x = 22 - 17.5 = 4.5
Langkah 4: Verifikasi
Cek ke persamaan (1): 2(4.5) + 3(3.5) + 3.5 = 9 + 10.5 + 3.5 = 23 ‚úì
Cek ke persamaan (2): 4.5 + 2(3.5) + 3(3.5) = 4.5 + 7 + 10.5 = 22 ‚úì
Cek ke persamaan (3): 3(4.5) + 3.5 + 2(3.5) = 13.5 + 3.5 + 7 = 24 ‚úì
Jawaban: Kue A memerlukan 4.5 kg bahan, Kue B memerlukan 3.5 kg bahan, dan Kue C memerlukan 3.5 kg bahan.
